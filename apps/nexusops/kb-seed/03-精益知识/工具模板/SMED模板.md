# SMED（单分钟快速换模）模板

SMED（Single Minute Exchange of Die）由新乡重夫（Shigeo Shingo）首创、丰田推广，是缩短换型停机时间、提升可用率（Availability）的核心方法。目标是把换模控制在**个位数分钟（<10min）**内，应对多品种小批量。

## 核心思想：区分内外部作业

| 作业类型 | 定义 | 典型动作 |
|----------|------|----------|
| **内部作业（IED）** | 必须停机才能做 | 拆旧模、装新模、调参数、试切首件 |
| **外部作业（OED）** | 运行时可提前做 | 预热新模、备工具、备料、打印参数单 |

> SMED 的精髓：**把尽可能多的内部作业转为外部作业**，停机时间只留真正必须停机的动作。

## 五步改善法

### 第 1 步：观察记录现状
- 全程录像换模过程（从上一产品最后一件 → 下一产品第一件合格品）
- 记录每个动作、耗时、操作人数、工具路径
- 绘制"换模作业时间表"（动作序列 + 内/外部标注）

### 第 2 步：区分内/外部作业
- 逐动作判定：停机才能做（内） vs 运行时可做（外）
- **立即收益**：通常可减少 30-40% 停机时间，无需任何技术投入

### 第 3 步：消除内部作业的浪费
对必须停机的动作，用以下手段压缩：
- **并行作业**：多人同时操作不同部位（而非串行）
- **标准化紧固件**：统一螺钉头尺寸，减少工具切换
- **快速夹紧**：用触摸式夹钳/旋转一次锁定替代螺栓
- **模具定位基准**：设计定位销，消除对中调试
- **工具定置**：工具车按换模顺序摆放，伸手即取

### 第 4 步：内部作业转外部
- 细分停机作业，分析每个步骤能否提前准备：
  - 新模具提前运到机台旁预热
  - 参数单提前打印、提前录入设备
  - 通知质检提前到位，缩短首件检验等待

### 第 5 步：消除调整
- 调整占整个换模时间的 **50-70%**，是最大损耗
- 通过**模具高度标准化 + 参数预设 + 防错设计**，使首件即合格，免去反复调试

## 换模点检表（换模前必查）

| 检查项 | 责任人 | 完成打勾 |
|--------|--------|----------|
| 新模具已运至机台旁 | 物料员 | ☐ |
| 模具已预热至工艺温度 | 工艺员 | ☐ |
| 换模工具齐备（按定置图） | 班长 | ☐ |
| 新产品参数单已打印 | 工艺员 | ☐ |
| 首件检验量具已就位 | 质检员 | ☐ |
| 上一产品末件已送检 | 质检员 | ☐ |

## 诊断动作

当 `schedule.changeover` 显示换模超时（>60min）时：

1. 录像换模过程 → 绘制动作时间表
2. 标注内/外部作业 → 计算外部作业占比
3. 外部作业占比 <30% → 优先"内转外"（见效快）
4. 内部作业有大量调整 → 优先消除调整（第 5 步，收益大但需投入）
5. 设定目标：换模时间缩短 50%

| 验证效果 | `schedule.ct_vs_takt` | 换模改善后 CT 是否回落 |

## 推动原则

- **团队作战**：需生产（实操）+ 生技（治具）+ 质量（良率检验）+ 安全协同，非单部门能成
- **先播种后收割**：从积极者班组试点，用数据示范再全推（见 `04-人与组织/L01产线负责人与阻力.md`）
- **数据验证**：改善后用 OEE / 换模时间客观数对比，而非主观感受
