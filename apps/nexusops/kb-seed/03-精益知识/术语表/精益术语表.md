# 精益术语表

本企业对精益术语的**定制化定义**（可能与教科书略有差异，以本文为准）。

## 核心指标

- **OEE**：见 `01-现场状态/OEE计算口径.md`
- **FPY（首次合格率）**：一次过线合格率，不含返工后合格
- **CT（Cycle Time）**：完成一个产品的实际耗时
- **Takt**：客户需求节奏 = 可用时间 / 需求量
- **MTBF**：平均故障间隔（Mean Time Between Failures）
- **MTTR**：平均修复时间（Mean Time To Repair）

## 损失分类

- **六大损失**：故障、换模、小停机、降速、缺陷、启动损失
- **可用率损失**：故障 + 换模 + 缺料待机 + 计划保养
- **性能损失**：小停机 + 降速 + 空转
- **质量损失**：报废 + 返工 + 降级

## 工具方法

- **5S**：整理/整顿/清扫/清洁/素养
- **SMED**：单分钟换模（Single Minute Exchange of Die），目标 <10min
- **VSM**：价值流图（Value Stream Mapping）
- **5Why**：连续问 5 个为什么找根因
- **5M1E**：人/机/料/法/环/测（鱼骨图分类）
- **A3**：单页报告（丰田标准），结构化问题解决
- **PFMEA**：过程失效模式与影响分析
- **SPC**：统计过程控制（控制图）
- **Cp/Cpk**：过程能力指数，Cpk ≥ 1.33 为合格

## 本企业特有

- **健康分**：融合 IoT 信号（振动/温度/电流）的设备综合健康评分（0-1）
- **场景 id**：normal / anomaly / crisis，用于工具调用切换 mock 数据
