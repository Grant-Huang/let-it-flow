# 案例：换模超时致可用率损失

## 问题现象

L01 注塑产线 OEE 从 82% 跌至 61%（anomaly 场景），其中**可用率从 92% 跌至 78%**，是最大损失项。当班换模时间累计 **180 分钟**（标准应 <60min），直接拖垮可用率。

## 诊断过程

1. `oee.decompose`：损失瀑布显示可用率损失占总损失的 52%（主因），性能率次之
2. `equipment.downtime`：停机事件中"换模超时"占 60min、"模具卡死"占 45min
3. `schedule.changeover`：当日换模 180min，远超 60min 标准
4. `schedule.bottleneck_resource`：瓶颈为"注塑机#1"
5. 现场观察：换模工具散乱、无定置，操作员边找工具边换模（串行）

## 根因（5Why）

| 层次 | 问 | 答 |
|------|----|----|
| Why1 | 为什么换模超时？ | 换模时间长（180min） |
| Why2 | 为什么换模时间长？ | 大量时间花在找工具和调试对中 |
| Why3 | 为什么找工具耗时？ | 工具无定置，散落在工具柜 |
| Why4 | 为什么调试对中耗时？ | 模具无定位基准，靠目测对中 |
| Why5 | 为什么无定置无定位？ | 未推行 SMED，沿用老换模流程 |

**根因**：换模流程未标准化（无 SMED），内外部作业未区分，工具无定置，模具无定位基准。

## 对策

### 临时
- 工具按换模顺序定置摆放（工具车），减少寻找时间
- 换模前预打印参数单（外部作业前置）

### 永久（SMED 五步法）
1. 录像换模全过程，区分内/外部作业
2. 外部作业（备模、预热、备料）全部前置到停机前
3. 模具加装定位销（消除对中调试，水平1防错）
4. 紧固件标准化（统一螺钉头，减少工具切换）
5. 目标：换模 ≤ 30min（缩短 83%）

## 教训沉淀

- **换模损失是"隐形可用率杀手"**：容易被忽视，但累计损失巨大
- **内外部作业区分是第一步**：通常可立即减 30-40% 停机时间，零成本
- **定位基准比提速更重要**：消除调试（占换模 50-70%）收益最大
