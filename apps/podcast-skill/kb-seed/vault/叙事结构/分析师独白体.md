# 分析师独白体

## 适用场景

- 对某个事件或趋势的深度分析和解读
- 有明确立场和观点的观察性内容
- 需要在逻辑上层层递进说理的主题
- 适合商业、技术、政策等专业领域

## 五段骨架

1. **开场（1-2分钟）**
   - 开门见山：指出当前热点或现象
   - 问题定位：这件事为什么值得讨论？
   - 示例："最近 AI agent 架构成为争议焦点，几位顶级研究者的观点分化明显。这背后反映的是什么？"

2. **背景阐述（3-5分钟）**
   - 事件脉络：时间线、参与者、关键动作
   - 信息铺垫：听众需要了解的基础事实
   - 避免过度堆砌：选最有说服力的 2-3 条

3. **核心分析（8-12分钟）**
   - 拆解问题：从多个维度看（技术 / 商业 / 伦理）
   - 观点对比：不同立场的论据
   - 深度挖掘：这个分歧背后的根本矛盾是什么？

4. **个人观点（3-5分钟）**
   - 表达立场：作为制作者的看法
   - 给出理由：为什么相信这个观点
   - 诚实表达局限："我可能忽视的是..."

5. **收尾（1-2分钟）**
   - 回应开场：和第一段形成呼应
   - 行动提示：听众可以怎么做（阅读原文 / 自己追踪）
   - 温暖结尾："下期我们讨论..."

## 语气特征

- 沉静、冷静、思辨性强
- 用"我认为"而非"据说"
- 允许表达不确定性
- 段落间逻辑清晰，层层递进

## 模板开场

> "你好，这是[节目名]。今天我们聊一个最近争议很大的话题——[核心问题]。表面上看，这是关于[表象]的讨论，但我觉得根本问题在于[深层矛盾]。我会从三个角度拆开它。"
