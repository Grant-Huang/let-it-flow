# 口播字数公式

## 核心公式

**字数 = 目标分钟数 × 210**

例：30分钟播客 ≈ 6300字

## 容差范围

±5%（部分长度）

例：30分钟允许 5985-6615字

## 背景

- 210字/分钟 是中文口语播读速度的标准参考
- 考虑停顿、呼吸、强调的自然节奏
- 不同发音人可能 ±10%，但 210 是安全基准

## 应用规则

1. **生成初稿后立即校验**：计算总字数
2. **超出容差时修改策略**：
   - 过长（>6.5%）：删除冗余修饰词、合并短句
   - 过短（<4.5%）：补充案例、展开论述（不是堆砌）
3. **分段时也要检验**：各章节合计 = 总数

## 示例

| 目标 | 字数范围 |
|------|--------|
| 15分钟 | 3045-3195字 |
| 20分钟 | 4060-4260字 |
| 30分钟 | 5985-6615字 |
| 45分钟 | 8978-9423字 |
