# 单句长度规范

## 核心规则

**单句 ≤ 25字**

## 背景

- 口语最舒适的停顿点是 15-25字
- 超过 25字 的句子在口播时容易撑晕听众
- 超过 30字 的句子几乎不可能一口气自然念出

## 违例修正

### 问题句

"虽然这项技术在理论上有很多潜力，但实际应用中我们还需要考虑成本、伦理和监管等多个复杂因素，所以短期内可能不会看到大规模应用。"（59字）

### 改正方案

拆成多句，用连词连接：

"虽然这项技术理论上有潜力，但我们面临多重挑战。首先是成本。其次是伦理和监管。所以短期难以大规模应用。"

- 句1：13字 ✓
- 句2：12字 ✓
- 句3：8字 ✓
- 句4：9字 ✓
- 句5：11字 ✓

## 检验工具

生成稿件后，逐句数字数。超过 25字 的用逗号或句号拆开。

## 特例

- 列举时可用顿号承接，但每个项 ≤15字
- 引用他人言论时不改，但整体句仍需 ≤25字
